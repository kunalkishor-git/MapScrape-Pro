# 📍 MapScrape Pro

MapScrape Pro is a professional-grade B2B lead extraction tool. It allows users to scrape business contact information from **Google Maps** and **Google Search** results, including a "Deep Scrape" feature that visits social media profiles to find hidden contact details.

## ✨ Features
* **Google Maps Mode:** Extract leads based on location, keyword, and category.
* **Google Search Mode:** Search with specific B2B tags (e.g., "SaaS", "Agency").
* **Contact Extraction:** Automatically finds emails and phone numbers on business websites.
* **Social Media Scraper:** Detects and visits Facebook, Instagram, LinkedIn, and X profiles to find contact info.
* **Export to CSV:** Download all results in a clean spreadsheet format.

## 🛠️ Prerequisites
* **Python 3.8+**
* **Google Chrome** (The scraper uses Selenium to interact with live data).
* **ChromeDriver:** Handled automatically by Selenium in most environments.
